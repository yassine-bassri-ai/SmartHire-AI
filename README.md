# SmartHire AI

## Description

SmartHire AI est une plateforme intelligente d'aide au recrutement basée sur l'Intelligence Artificielle.

Le projet analyse automatiquement les CV, extrait les compétences, compare les profils aux offres d'emploi et prédit les candidats les plus adaptés grâce au Machine Learning et au Deep Learning.

## Objectifs

- Extraire le texte des CV PDF
- Prétraiter les données avec le NLP
- Identifier automatiquement les compétences
- Comparer les CV aux offres d'emploi
- Entraîner plusieurs modèles de Machine Learning
- Comparer les résultats avec DistilBERT
- Stocker les données dans MySQL
- Développer une API avec FastAPI
- Créer une interface avec Streamlit
- Construire un tableau de bord Power BI

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Transformers (DistilBERT)
- spaCy
- MySQL
- FastAPI
- Streamlit
- Power BI
- Git & GitHub
